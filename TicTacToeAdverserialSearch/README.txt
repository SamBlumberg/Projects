This project is divided into 3 folders, TTT, ATTT and STTT
Each program is a version of Tic Tac Toe where you play against the program which uses a form of 
adversarial search. 

TTT contains part 1; simple tic tac toe
It can be complied with 
	javac TTT.java
and ran with	
	java TTT

You can make moves by passing a single number 1-9 as seen here 
1 | 2 | 3
_________
4 | 5 | 6
__|___|__
7 | 8 | 9

-------------------------------------------------------------------------------------------


ATTT contains part 2; Advanced tic tac toe
It can be complied with 
	javac AdvancedTTT.java
and ran with	
	java AdvancedTTT

Advanced Tic Tac Toe is a 3x3 board of 3x3 boards. You win when a single board is won. 
You can make moves by entering 2 numbers seperated by a space. First which board you will
play on, and second the move within the board i.e. 2 5. 

----------------------------------------------------------------------------------------------------

STTT contains my extra credit: super tic tac toe
It can be complied with 
	javac SuperTTT.java
and ran with	
	java SuperTTT

Super Tic Tac Toe is a 3x3 board of 3x3 boards. You win when 3 small boards in a row are won. 
You can make moves by entering 2 numbers seperated by a space. First which board you will
play on, and second the move within the board i.e. 2 5.
